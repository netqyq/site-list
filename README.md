# Site List
Site List 具有常用网站导航功能和个人网站收藏功能。


* Ruby version
Ruby 2.3.1
Rails 5.0.0.1

* 主要用到Gems
```
devise
sunspot_rails
kaminari
cancan
rucaptcha
```
* Configuration
```
新建2个文件
config/secrets.yml
config/database.yml

```

* Database creation
```
后台数据库为PostgreSQL
rake db:create
rake db:migrate
```
